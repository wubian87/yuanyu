# 原语言（YuanYuYan）

> 人与AI协作时代的原语语言 —— 用最少的规则描述本质，让AI理解意图并生成一切实现。  
> 不写代码，不画流程，只需四层描述：信息、操作、呈现、边界。

**核心思想：知其所向，自得其路**

一句话：用原语描述本质，让AI生成一切

## 四层结构

| 层级 | 含义 | 问题 |
|------|------|------|
| 信息 | 是什么 | 系统管理什么数据？ |
| 操作 | 做什么 | 用户能做什么？ |
| 呈现 | 见什么 | 用户看到什么？ |
| 边界 | 守什么 | 什么要做/不做？ |

## 快速开始

```yaml
名称: 记账

信息:
  账单:
    金额: number
    类型: [收入, 支出]

操作:
  - 记一笔
  - 看流水

呈现:
  首页: 本月概览 + 快速记账

边界:
  守: [离线可用]
  未: [多人共享]
```

## 入口

学原语 → 写四层 → 交给AI → 看它生
