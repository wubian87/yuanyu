# 原语言（YuanYu）

> 用原语描述本质，让AI生成一切

## 这是什么

原语言是人与AI协作时代的原语语言。

传统编程：你告诉机器每一步怎么做  
原语言：你描述它是什么，AI知道怎么做

## 核心思想

**知其所向，自得其路**

你不是建筑师画图纸，你是农人种种子。  
AI是阳光，场景是土壤，生长自然发生。

## 四层结构

| 层 | 问题 | 说明 |
|---|------|------|
| 信息 | 是什么 | 数据的骨架 |
| 操作 | 做什么 | 可能的动作 |
| 呈现 | 见什么 | 界面的形态 |
| 边界 | 守什么 | 规则与可能 |

## 快速示例

```yaml
名称: 记账

信息:
  账单:
    金额: number
    类型: [收入, 支出]
    分类: [餐饮, 交通, 购物, 娱乐]
    时间: datetime

操作:
  - 记一笔
  - 看流水
  - 查统计

呈现:
  首页: 本月概览 + 快速记账
  流水: 按日列表
  统计: 饼图

边界:
  守: [离线可用, 数据本地存储]
  未: [多人共享, 预算提醒]
```

## 开始使用

```
学原语 → 写四层 → 交给AI → 看它生
```
